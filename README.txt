Project Title : VahanVikraya
Instruction to run the project:

    1) Dependencies installation step

        Run below command in the ApniCar folder
        $ npm install

        Run below command in the client folder inside ApniCar folder
        $ npm install

    2) Running the project

        Run below command in the ApniCar folder
        $ npm start

        Run below command in the client folder inside ApniCar folder
        $ npm start
